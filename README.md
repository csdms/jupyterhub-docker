# csdms-jupyterhub

A Docker image for setting up a JupyterHub,
based on the [scipy-notebook][scipy-notebook] image,
with [CSDMS Workbench][csdms-workbench] software
(Landlab, Basic Model Interface, Data Components) pre-installed.
